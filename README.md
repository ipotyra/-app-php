# app-php

A project built during a PHP class to practice creating dynamic pages, simulating the rooms of a house.

## About

The app displays different PHP pages, each representing a room in a house, with illustrative images and a simple login system.

## Structure

```
app-php/
├── index.php          # Home page
├── login.php          # Login page
├── sala.php            # Living room page
├── cozinha.php         # Kitchen page
├── quarto.php          # Bedroom page
├── banheiro.php         # Bathroom page
├── imagens/             # Images used across the pages
│   ├── sala.jpg
│   ├── cozinha.jpg
│   ├── quarto.jpg
│   ├── quarto2.jpg
│   └── banheiro.jpg
└── peimeiroarquivo.py   # Python study/test file
```

## How to run

1. Place the project on a local server with PHP support (e.g. XAMPP).
2. Start Apache.
3. Open `http://app-php.local` (or your configured address) in the browser.

## Status

Class project, made for learning purposes, no ongoing maintenance planned.
